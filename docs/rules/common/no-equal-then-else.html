No equal then else

Rule id#

no-equal-then-else

Severity#

Warning

Description#

Warns when if statement has equal then and else statements or conditional expression has equal then and else expressions.

Example#

Bad:

final value1 = 1;final value2 = 2;
...
// LINTif (condition) {  result = value1;} else {  result = value1;}
...
result = condition ? value1 : value1; // LINT

Good:

final value1 = 1;final value2 = 2;
...
if (condition) {  result = value1;} else {  result = value2;}
...
result = condition ? value1 : value2;
